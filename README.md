# 无人点餐系统webapp(Vue)

> entry: http://localhost:8080/#/start?roomid=b040

## 安装步骤

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
